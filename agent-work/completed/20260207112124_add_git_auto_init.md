# Add Git Auto Init

## Status: completed 20260207112254

## Context
Push command fails if synced-files folder isn't a git repo. Should auto-init if .git doesn't exist.

## Value Proposition
- One-time setup handled automatically
- No manual `git init` needed
- Idempotent - safe to run multiple times

## Alternatives considered
- Manual init: Burden on user, easy to forget
- Init on every command: Wasteful checks
- **Check + init when needed (chosen)**: Efficient, user-friendly
