# Add CLI Commands

## Status: completed 20260207104212

## Context
CLI needs subcommands for file tracking operations and git repo management.

## Value Proposition
- `track <files...>` - Add files to sync config
- `untrack <files...>` - Remove files from sync config
- `pull` - Git pull in synced-files folder
- `push` - Git push in synced-files folder
- `set-origin-repo <url>` - Set git remote origin

## Alternatives considered
- Separate binary per command: Overkill, adds complexity
- Single flag-based (`--track`, `--untrack`): Harder to extend, less idiomatic for Go CLI
- **Subcommands with flag package**: Standard Go CLI pattern, clean, extensible
